# Tic-Tac-Toe
AI Tic Tac Toe Game
utilizes 4 main components 
GameBoard: state managemnet
AIPlayer: Q-learning agent with persistant memory(stores between sessions)
MoveHistory: tracts your patterns so AI can counter your tendicies
UI: game display + stats/learning progress

the idea is The agent gradually learns which moves from which positions lead to good outcomes against you specifically and learns to counter you.
